What is an UNSAT Core?
An unsatisfiable core is a minimal subset of constraints that are sufficient to make the problem unsatisfiable. This helps identify which constraints conflict.

Problem: Contradictory Constraints

(set-logic QF_UF)
(declare-const p Bool)
(assert p)        ; p must be true
(assert (not p))  ; p must be false (contradiction!)
(check-sat)
